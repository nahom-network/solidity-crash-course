## Solidity Arrays

Working with arrays in Solidity can be **tricky**! <emoji id="confounded" />

We'll need to understand the **data location**, or where the array is stored. <emoji id="round_pushpin" />

We'll also need to understand the **memory allocation**, whether it is fixed size or it can be dynamically resized. <emoji id="brain" />

Don't you worry! We'll bring these concepts to **light** in this coding tutorial. <emoji id="bulb" />
